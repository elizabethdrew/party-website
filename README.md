# elizabethdrew.github.io

## Run locally

### Option 1: Quick static preview (recommended)

From the project root:

```bash
python3 -m http.server 4000
```

Then open:

`http://localhost:4000`

### Option 2: GitHub Pages/Jekyll preview

If you want to preview with Jekyll behavior:

```bash
gem install bundler jekyll
jekyll serve --livereload
```

Then open:

`http://localhost:4000`
